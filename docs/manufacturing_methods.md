# 나노 디퓨저 + 체크보드 위상막 — 제조 방법 종합 분석

> 나노 디퓨저를 위상변조막에 결합하면 **주기적 회절 억제 + 산란 스펙트럼 화이트닝**을 동시에 달성할 수 있어 특허성이 매우 높다.

---

## 제조 방법 비교

### 방법 A: 중공 입자 + 현상 제거 방식

**원리**: 포지티브 PR에 중공 실리카(hollow SiO₂)를 분산 → UV 노광 → 현상 시 입자 제거 → 보이드 잔류

- 장점: 단일 공정, 비용 저렴
- 단점:
  - 입자 제거 수율 불균일 (depth direction)
  - 보이드 크기 제어 어려움
  - 잔류 입자가 광 흡수/산란 불균일 유발
- **현실 평가: ⚠️ 권장하지 않음** — 제어성이 낮고 재현성 문제

### 방법 B: 희생 나노스피어 + Lift-off ⭐ 권장

**원리**: PS(polystyrene) 또는 PMMA 나노스피어를 하층에 자기조립 → 위상막 증착 → 용매로 희생층 제거 → 주기적/랜덤 디봇 패턴

- 장점: 입자 크기 정밀 제어 (50-500nm), 재현성 우수
- 적용: L'Oréal, 3M 광학 필름 양산 실적
- **추천도: ⭐**

### 방법 C: 나노임프린트 리소그래피 (NIL) ⭐⭐ 최고 권장

**원리**: e-beam으로 마스터 몰드 제작 → 블루레이 NIL 수지 도포 → UV 경화 임프린트 → 이형

- 장점:
  - 완전 결정론적 랜덤 패턴 (재현성 100%)
  - 서브-100nm 해상도
  - 대면적 가능 (SDC/LGD 양산 호환)
- 단점: 초기 마스터 투자비
- **추천도: ⭐⭐ (최고 권장)**

### 방법 D: Block Copolymer Self-Assembly (BCP)

**원리**: PS-b-PMMA BCP 코팅 → 열 어닐링 → 자기조립 패턴 → 선택적 에칭

- 장점: 15-50nm 해상도, 대면적
- 단점: 결함 밀도 관리 필요

### 방법 E: 레이저 스페클 노광

**원리**: 가간섭성 레이저 스페클 패턴으로 포토레지스트 직접 노광

- 장점: 단일 공정, 진짜 랜덤
- 단점: 삼성 기존 설비 미보유

## 최적 제조 공정 (특허 출원용)

### 복합 공정: 방법 C (NIL) + 체크보드 위상 패턴 통합

**Step 1. BM 층 형성**
- Cr/MoOx 스퍼터링, 1.5μm 기본 두께
- 기판: 유리 또는 PI 플렉시블

**Step 2. 나노입자 분산 PR 도포**
- 중공 SiO₂ 나노입자 (150nm, 8vol%) 분산
- 스핀코팅 또는 슬릿코팅

**Step 3. NIL 임프린트**
- e-beam 마스터 몰드: 체크보드 패턴 + 랜덤 나노 디봇
- UV 경화 임프린트
- 이형

**Step 4. 에칭 (두께 변조)**
- 체크보드 패턴에 따라 BM 두께 변조
- Δd = λ/(2(n-1)) ≈ 344nm (λ=550nm, n=1.8)

**Step 5. PR 제거 + 보호막**
- 잔류 PR 스트립
- SiNx 또는 SiO₂ 보호막 증착

## 특허 청구항 골격 (예시)

### 독립항 1 (구조)
유기 발광 표시 장치에 있어서, 복수의 화소 사이에 배치된 블랙 매트릭스(BM)를 포함하되,
상기 BM은 인접하는 화소 간에 교대로 상이한 두께를 가지는 체크보드 위상 변조 구조와,
상기 BM 내에 분산된 랜덤 나노 디퓨저 구조를 동시에 포함하는 것을 특징으로 하는 표시 장치.

### 독립항 2 (제조 방법)
블랙 매트릭스(BM) 층을 형성하는 단계;
나노입자가 분산된 감광성 수지를 도포하는 단계;
나노임프린트 리소그래피(NIL)로 체크보드 패턴과 랜덤 나노 구조를 동시에 임프린트하는 단계;
에칭으로 BM 두께를 변조하는 단계;
를 포함하는 표시 장치의 제조 방법.

### 종속항 예시
- 상기 두께 차이 Δd는 λ/(2(n-1)) (n: BM 굴절률, λ: 가시광 중심 파장)
- 상기 나노 디퓨저는 직경 50-300nm의 중공 실리카 입자
- 상기 체크보드 패턴은 인접 화소 간 π 라디안의 위상차를 부여
