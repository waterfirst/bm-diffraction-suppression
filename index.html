BM 회절 억제 연구 -- 반사회절 패턴 분석

원편광판 없이 OLED 외부광 반사 회절을 억제하는 체크보드 위상 + 랜덤 나노 디퓨저

📋 연구 개요

⚠️ Problem

원편광판(CPL)이 OLED 발광의 50%를 흡수하여 디스플레이 수명을 단축시키고 전력 효율을 저하시킵니다. 그러나 CPL을 제거하면 BM 격자에 의한 무지개색 회절 패턴이 외부 반사광에서 관찰됩니다.

✅ Solution

체크보드 위상 변조와 랜덤 나노 디퓨저를 BM 구조에 적용하여, 원편광판 없이도 1차 회절을 -25 dB 이상 억제합니다. 에지 아포다이제이션을 추가하면 고차 회절까지 제어됩니다.

⚫

체크보드 위상

인접 픽셀에 π 위상차를 부여하여 1차 회절의 destructive interference를 유도합니다.

🔬

나노 디퓨저

SiO₂ 중공 나노입자(150nm, 8vol%)를 분산하여 잔여 회절 에너지를 넓은 각도로 분산시킵니다.

🎯

에지 아포다이제이션

BM 경계를 raised-cosine 프로파일로 점진적 변화시켜 sinc 사이드로브를 억제합니다.

📊 물리 수식

Unit Cell 투과/반사 함수 t(x, y) = rect(x/a) · rect(y/b) · exp(iφ(x,y))
체크보드 위상 변조 φ(m, n) = π · [(m + n) mod 2]
1차 회절 완전 소멸 (Destructive Interference) Etotal = E₀ · [1 + exp(iπ)] = E₀ · [1 − 1] = 0
최적 BM 두께 변조 Δd = λ / 2(n − 1)
예시: λ=550nm, n=1.8 → Δd ≈ 344 nm
2D 주기 격자 T(x,y) = Σm Σn t(x − m·px, y − n·py)
Fourier 변환 (회절 패턴) I(kx, ky) = |FT{T(x,y)}|²
sinc 함수 형태 FT{rect(x/a)} = a · sinc(a · kx)
회절 차수 위치 kx,m = m · λ / px
ky,n = n · λ / py
랜덤 디퓨저 효과 φ(m,n) = φcheckerboard(m,n) + δmn
δmn ~ Uniform(0, σ·2π)
Ipeak ∝ 1/N   (N = 디퓨저 랜덤 요소 수)
에지 아포다이제이션 w(x) = 0.5 · [1 + cos(π · (|x| − a/2 + β) / β)]
a/2 − β < |x| < a/2, β = 아포다이제이션 폭
광대역 평균 피크 강도 ⟨Ipeak⟩(λ) ≈ I₀(λ) · sinc²(Δd·(n−1)/λ − 1/2) + Iscatter(σ)

🏭 제조 공정

최적 복합 공정: NIL + 체크보드 위상 패턴 통합 (5단계)

1

BM 층 형성

Cr/MoOx 스퍼터링으로 1.5μm 기본 두께의 BM 층을 유리 또는 PI 플렉시블 기판 위에 형성합니다.

2

나노입자 분산 PR 도포

중공 SiO₂ 나노입자(150nm, 8vol%)를 분산한 감광성 수지를 스핀코팅 또는 슬릿코팅합니다.

3

NIL 임프린트

e-beam 마스터 몰드로 체크보드 패턴 + 랜덤 나노 디봇을 UV 경화 임프린트합니다.

4

에칭 (두께 변조)

체크보드 패턴에 따라 BM 두께를 변조합니다. Δd = λ/(2(n−1)) ≈ 344nm

5

PR 제거 + 보호막

잔류 PR을 스트립하고 SiNx 또는 SiO₂ 보호막을 증착하여 공정을 완료합니다.

방법 해상도 재현성 양산성 비용 추천
A. 중공 입자 100-300nm 낮음 보통 저 ⚠️
B. 희생 나노스피어 50-500nm 우수 우수 중 ★
C. NIL <100nm 100% 우수 중-고 ★★
D. BCP 15-50nm 보통 보통 중 —
E. 레이저 스페클 200-1000nm 보통 낮음 고 —

A. 중공 입자 + 현상 제거 방식

포지티브 PR에 중공 실리카(hollow SiO₂)를 분산하여 UV 노광 후 현상 시 입자를 제거하고 보이드를 잔류시킵니다. 단일 공정으로 비용이 저렴하나, 입자 제거 수율 불균일과 보이드 크기 제어 어려움 때문에 권장하지 않습니다.

B. 희생 나노스피어 + Lift-off

PS 또는 PMMA 나노스피어를 자기조립 후 위상막을 증착하고 용매로 희생층을 제거합니다. 입자 크기 정밀 제어(50-500nm)가 가능하고, L'Oreal/3M 양산 실적이 있어 재현성이 우수합니다.

C. 나노임프린트 리소그래피 (NIL) -- 최고 권장

e-beam 마스터 몰드 제작 후 UV 경화 임프린트. 완전 결정론적 랜덤 패턴(재현성 100%), 서브-100nm 해상도, SDC/LGD 양산 호환 가능. 초기 마스터 투자비가 필요하나 최적 솔루션입니다.

D. Block Copolymer Self-Assembly (BCP)

PS-b-PMMA BCP 코팅 후 열 어닐링으로 자기조립 패턴을 형성합니다. 15-50nm 초미세 해상도가 가능하나 결함 밀도 관리가 필요합니다.

E. 레이저 스페클 노광

가간섭성 레이저 스페클 패턴으로 포토레지스트를 직접 노광합니다. 단일 공정의 진짜 랜덤 패턴이나, 삼성 기존 설비 미보유로 도입 장벽이 있습니다.
